Animaciones

Son animaciones que no requieren un evento o un cambio de estado para ejecutarse

Para crear una animacion se requiere 2 pasos basicos

  • 1.Definir la animacion en el elemento
  • 2.Crear los keyframes

La animacion requiere como minimo el nombre y duracion

animation:girar 2s;

Se le pueden pasar mas valores.Por ejemplo: infinite para que nunca se detenga,alternate para que se anime de ida y vuelta o linear para que la velocidad sea constante

animation:girar 2s infinite alternate linear;

Los keyframes determinan los momentos en los que los estilos cambian.lo mas facil es poner from {} y to {} Pero tambien se podrian poner Kyeyframes especificos con porcentajes.

@keyframes girar{ 0%{ color:red; } 50%{ color:green; } 100%{ color:blue; } }
